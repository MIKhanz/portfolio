Core Areas

Innovation Portfolio

01
Mirror Beacon Technology

Breakthrough approach using existing BLE and Wi-Fi infrastructure to create virtual beacon networks — eliminating dedicated hardware while achieving high-accuracy indoor positioning. Enables scalable, cost-effective deployment across any building type.

02
Offline Positioning Algorithms

Novel compression and quality monitoring techniques enabling precise indoor navigation on low-memory devices without server connectivity. Supports wearables, IoT sensors, and offline-first mobile applications in restricted network environments.

03
Multi-Floor Alignment

Patented methods for accurate vertical positioning and floor transition detection in multi-storey structures. Resolves the fundamental challenge of Z-axis navigation in complex venues such as hospitals, airports, and shopping centres.

04
Radio Map Data Structures

Efficient storage and retrieval architectures for large-scale indoor radio fingerprint databases. Enables fast, accurate positioning queries at scale while minimising bandwidth and storage footprint on device and server.

05
Visual Positioning Systems

Camera-based indoor localisation using computer vision and machine learning — delivering precise positioning without dedicated sensors. Designed for automotive companion apps and large-venue navigation without infrastructure investment.

06
Geofence Trigger Logic

Intelligent, low-power geofence detection algorithms that minimise battery drain while maintaining real-time responsiveness. Applicable to industrial safety zones, marketing activation, access control, and consumer location services.
